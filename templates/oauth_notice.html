  2. 2 Set the Authorized Redirect URI to:
    http://127.0.0.1:5000/oauth/google/callback
  3. 3 Install Flask-Dance: pip install flask-dance and add your GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET to the app config.
    4. {% else %}

  5. 2 Set the Authorization Callback URL to:
    http://127.0.0.1:5000/oauth/github/callback
  6. 3 Install Flask-Dance: pip install flask-dance and add your GITHUB_OAUTH_CLIENT_ID and GITHUB_OAUTH_CLIENT_SECRET to the app config.
